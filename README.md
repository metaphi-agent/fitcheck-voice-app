# fitcheck

> Voice-first fashion assistant - your best fit, every time

## Overview

fitcheck is a voice-powered web app that helps you find the perfect outfit for any occasion. Simply speak or select from example prompts to get instant, context-aware fashion recommendations.

## Features

- **Voice Recognition**: Real-time speech-to-text using Web Speech API
- **Smart Recommendations**: Context-aware outfit suggestions for different occasions
- **Beautiful Animations**: Smooth transitions and pulse effects with Framer Motion
- **Mobile-First Design**: Optimized for iPhone 16 Pro (402x874px)
- **Quick Actions**: Three pre-configured prompts for instant recommendations

## Tech Stack

- React 19 + TypeScript
- Vite 6 (fast builds)
- Tailwind CSS 4 (utility-first styling)
- Framer Motion (animations)
- Lucide React (icons)
- Web Speech API (voice recognition)

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Usage

1. **Tap the microphone button** to start voice input
2. **Speak your occasion** (e.g., "Going to a dinner date")
3. **Get instant recommendations** for tops, bottoms, and footwear

Or click one of the example prompts:
- "Going to a dinner date"
- "Meeting friends for brunch"
- "Heading to work"
